## Getting Started

Required to install: [Node.js](https://nodejs.org/)

Install dependencies:

```bash
bun install
```

Run Development

```bash
bun dev
```

---

> [!NOTE]
> if not using `bun` just delete `bun.lockb`, and switch to another package manager like:

```bash
# npm install
# yarn install
# pnpm install
```
